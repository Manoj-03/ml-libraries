# ML Libraries

Practice notebooks and datasets covering core Python libraries used in data analysis and machine learning workflows.

## Contents

### `matplotlib/`
Plotting and data visualization basics — single and multiple plots.

### `numpy/`
Array operations, indexing, and numerical computing fundamentals.

### `pandas/`
DataFrame operations, data manipulation, and analysis.

### `practice/`
Hands-on data cleaning and exploratory data analysis (EDA) notebooks using real-world datasets.

### `sample_data/`
Datasets used across the notebooks (CSV/JSON) — e.g. Olympic athlete events, IMDB movies, e-commerce purchases, salaries.

### `seaborn_practice/`
Statistical data visualization using Seaborn.

## Tech Stack
- Python
- Jupyter Notebook
- NumPy, Pandas, Matplotlib, Seaborn

## Setup
```bash
pip install numpy pandas matplotlib seaborn
jupyter notebook
```
